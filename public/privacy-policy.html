Kebijakan Privasi

Kebijakan Privasi ini menjelaskan bagaimana aplikasi Catatan Digital mengumpulkan, menggunakan, dan melindungi informasi pengguna.

1. Informasi yang Kami Kumpulkan

Aplikasi ini tidak mengumpulkan data pribadi pengguna. Semua catatan disimpan secara lokal di perangkat pengguna menggunakan teknologi penyimpanan browser (local storage).

Kami tidak mengumpulkan:

  • Nama
  • Email
  • Nomor telepon
  • Lokasi
  • Kontak
  • Informasi perangkat

2. Penggunaan Informasi

Karena aplikasi tidak mengumpulkan data pribadi, maka tidak ada data pengguna yang diproses, disimpan di server, atau dibagikan kepada pihak ketiga.

3. Penyimpanan Data

Semua data catatan disimpan secara lokal di perangkat pengguna. Data akan tetap berada di perangkat pengguna kecuali dihapus secara manual oleh pengguna atau saat aplikasi dihapus.

4. Keamanan Data

Kami menjaga agar aplikasi tidak mengakses atau mengirimkan data ke server eksternal, sehingga privasi pengguna tetap terjaga.

5. Perubahan Kebijakan Privasi

Kebijakan Privasi ini dapat diperbarui sewaktu-waktu. Perubahan akan ditampilkan di halaman ini.

6. Kontak

Jika Anda memiliki pertanyaan terkait kebijakan privasi ini, silakan hubungi:

Email: support@catatkan.app
